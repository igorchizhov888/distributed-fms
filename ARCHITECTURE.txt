EVENT FLOW (Multi-Protocol):
1. Multiple sources → Adapters (SNMP, gNMI, Future)
2. Adapters → Kafka (unified bus)
3. EventConsumer → Ignite (distributed processing)
4. Correlation Engine → Parent/Child grouping
5. Ignite Cache → Real-time subscriptions
6. gRPC-Web → Browser UI

SCALABILITY:
- Horizontal: Add more Ignite nodes
- Protocol: Add new adapters without changing core
- Throughput: Thousands of events/sec per adapter
